🛒 User-friendly Grocery Shopping Website
🔐 Authentication & Role-based Access (Admin, User, Delivery Partner)
📦 Admin Panel to manage products, orders & delivery partners
🚴 Delivery Partner Dashboard with live location sharing
📍 Real-time Order Tracking System
📧 Automated Emails (offers, low stock alerts)
⚡ Auto-assign delivery partners using background jobs

🛠️ Tech Stack Used:

Frontend: React JS, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL (Neon Database)
Background Jobs: Inngest
